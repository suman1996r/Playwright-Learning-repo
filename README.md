# Playwright Learning

This repository contains Playwright example scripts used while learning Playwright.

Structure
- Chapter_01_Basics/: basic examples and setup verification scripts.

Quick start
1. Install Node.js (v16+ recommended).
2. From the repo root, install Playwright Test:

```bash
npm install -D @playwright/test
npx playwright install
```

3. Run tests/examples:

```bash
npx playwright test
```
